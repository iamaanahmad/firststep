# FirstStep Demo dApp

Sample Solana dApp showcasing FirstStep SDK features.

## Development

```bash
pnpm dev
```

Runs on `http://localhost:3000`

## Features

- Hero page with "Try as Guest" vs "Connect Wallet" options
- Feature demo (e.g., token swap, NFT mint)
- Gas sponsorship badge
- Transaction history with sponsorship indicators
